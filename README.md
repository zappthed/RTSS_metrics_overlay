# RTSS_metrics_overlay

A custom hardware metrics overlay for RivaTuner Statistics Server's Overlay Editor that can be used to monitor the hardware of the PC.

## Overview

A simple yet functional RTSS overlay that provides vital metrics ensuring proper insights during gameplay or stress tests.
The overlay uses joint data from RTSS and PresentMon built-in plugin that build the several metrics to monitor like CPU, GPU, and many more.
## Features

| Hardware | Data |
| - | - |
| CPU | Usage(%), Clock(MHz), Power(W), Temp° |
| RAM (Total RAM) | Usage(%) |
| GPU | Usage(%), Clock(MHz), Power(W), Temp° |
| VRAM (Total VRAM) | Usage(%), Clock(MHz), Temp° |
